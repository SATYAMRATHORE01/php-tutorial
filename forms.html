PHP Forms (Form Handling & Validation)

Forms are an essential part of web applications. PHP allows you to create, handle, and validate forms. Let's explore how to work with forms in PHP:

Creating a Form

You can create HTML forms using the <form> element. For example, creating a simple login form:

<form action="login.php" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>
    
    <button type="submit">Login</button>
</form>
?>

Handling Form Submissions

PHP can process form data submitted via the $_POST or $_GET superglobals. For example, handling the login form submission in a PHP script:

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    $password = $_POST["password"];

    // Perform authentication and validation
}
?>

Form Validation

Form validation ensures that the data submitted is valid and safe to process. You can use PHP to validate form fields, check for required values, and sanitize input. For example, validating the username:

<?php
$username = $_POST["username"];
if (empty($username)) {
    $errors[] = "Username is required.";
} elseif (!preg_match("/^[a-zA-Z0-9_]+$/", $username)) {
    $errors[] = "Invalid username format.";
}
?>

Displaying Validation Errors

If validation fails, you can display error messages to the user. For example, showing validation errors in the login form:

<?php
if (!empty($errors)) {
    echo "<div class='alert alert-danger'>";
    echo "<p>Please correct the following errors:</p>";
    echo "<ul>";
    foreach ($errors as $error) {
        echo "<li>" . $error . "</li>";
    }
    echo "</ul>";
    echo "</div>";
}
?>

Processing Form Data

After validation, you can process and use the form data. For example, performing authentication and redirection after a successful login:

<?php
if (isset($_POST["username"]) && isset($_POST["password"])) {
    // Check username and password
    if ($username === "user" && $password === "password") {
        header("Location: welcome.php");
        exit();
    } else {
        $errors[] = "Invalid credentials.";
    }
}
?>

Working with forms in PHP is essential for building interactive web applications. Proper form handling and validation are crucial for user data security and the overall user experience.
