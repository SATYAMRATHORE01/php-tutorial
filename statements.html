PHP Statements

In PHP, statements are the building blocks of your code. Statements are executed sequentially and can perform various tasks. Here's an overview of different types of PHP statements:

Assignment Statements

Assignment statements are used to assign values to variables. For example:

<?php
    $name = "John";
    $age = 30;
?>

Conditional Statements

Conditional statements allow you to execute code based on conditions. Common types include if, else, and switch. For example:

<?php
    $age = 25;
    if ($age >= 18) {
        echo "You are an adult.";
    } else {
        echo "You are not an adult.";
    }
?>

Loop Statements

Loop statements allow you to repeat code. Common types include for, while, and foreach. For example:

<?php
    for ($i = 1; $i <= 5; $i++) {
        echo "Iteration $i<br>";
    }
?>

Control Statements

Control statements alter the normal flow of a program. For example, the break statement can exit a loop prematurely, and the continue statement can skip the current iteration. For example:

<?php
    for ($i = 1; $i <= 5; $i++) {
        if ($i == 3) {
            break;
        }
        echo "Iteration $i<br>";
    }
?>

Statements are essential for controlling the flow and behavior of your PHP code. Whether it's assigning values to variables, making decisions, or repeating actions, statements are fundamental to creating dynamic web applications.
