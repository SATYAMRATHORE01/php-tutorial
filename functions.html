PHP Functions

In PHP, functions are a powerful feature that allow you to encapsulate code for reuse and modularity. PHP functions can be categorized into several types:

Simple Functions (No Return Value):

Simple functions in PHP perform a specific task but do not return a value. They are called for their side effects, such as displaying information.


function greet($name) {
    echo "Hello, $name!";
}

// Call the function
greet("John");
Functions with a Return Value:

These functions return a value to the caller, which can be stored in a variable or used for further computation.


function add($a, $b) {
    return $a + $b;
}

// Call the function and store the result
$sum = add(5, 3);
Recursive Functions:

Recursive functions are functions that call themselves. They are commonly used for solving problems that can be broken down into smaller, similar subproblems.


function factorial($n) {
    if ($n <= 1) {
        return 1;
    }
    return $n * factorial($n - 1);
}

// Calculate the factorial of 5
$result = factorial(5);
Anonymous Functions (Closures):

Anonymous functions, also known as closures, allow you to create functions on the fly. They are useful for tasks like callbacks and event handling.


$add = function ($a, $b) {
    return $a + $b;
};

// Call the anonymous function
$result = $add(7, 2);
