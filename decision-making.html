Decision Making in PHP

In PHP, decision making is like making choices in real life. It allows your program to do different things based on different conditions. Here's how it works:

The "if" Statement

The "if" statement lets you ask a question and take different actions based on the answer. It's like deciding to go out if the weather is sunny:

<?php
    $weather = "sunny";
    if ($weather == "sunny") {
        echo "Let's go out!";
    } else {
        echo "Stay indoors.";
    }
?>

The "else if" Statement

If you have more than two options, you can use "else if" to make more decisions. For example, choosing an outfit based on the weather:

<?php
    $weather = "rainy";
    if ($weather == "sunny") {
        echo "Wear shorts.";
    } else if ($weather == "rainy") {
        echo "Grab an umbrella.";
    } else {
        echo "Decide what to wear.";
    }
?>

The "switch" Statement

When you have many options, a "switch" statement helps you choose between them. Think of it as picking a drink from a menu:

<?php
    $choice = "coffee";
    switch ($choice) {
        case "tea":
            echo "Tea is a good choice.";
            break;
        case "coffee":
            echo "Coffee is a classic.";
            break;
        default:
            echo "Make your selection.";
    }
?>

Decision making in PHP allows your program to act intelligently based on different conditions, just like how you make choices in everyday life. It's a way to add flexibility to your code.
