PHP Loops

Loops in PHP are used to execute a block of code repeatedly. They are a powerful tool for performing tasks that require repetitive actions. Let's explore different types of loops in PHP:

The "for" Loop

The "for" loop is used when you know how many times you want to repeat an action. It has a counter and continues as long as the condition is true. For example, printing numbers from 1 to 5:

<?php
    for ($i = 1; $i <= 5; $i++) {
        echo $i . "<br>";
    }
?>

The "while" Loop

The "while" loop is used when you want to repeat an action as long as a condition is true. It checks the condition before each iteration. For example, counting down from 5 to 1:

<?php
    $count = 5;
    while ($count > 0) {
        echo $count . "<br>";
        $count--;
    }
?>

The "do...while" Loop

The "do...while" loop is similar to the "while" loop, but it ensures that the code block is executed at least once. It checks the condition after each iteration. For example, prompting the user for input until a valid choice is made:

<?php
    $choice = "";
    do {
        echo "Choose an option (A, B, C): ";
        $choice = readline();
    } while ($choice != "A" && $choice != "B" && $choice != "C");
?>

The "foreach" Loop

The "foreach" loop is used for iterating over arrays. It simplifies array traversal and is particularly useful when working with arrays of data. For example, iterating through an array of names:

<?php
    $names = ["Alice", "Bob", "Charlie", "David"];
    foreach ($names as $name) {
        echo "Name: " . $name . "<br>";
    }
?>

Loops in PHP are a fundamental concept for automating repetitive tasks. Whether you know the number of iterations or need to keep looping until a certain condition is met, PHP provides versatile loop constructs to accomplish your goals.
