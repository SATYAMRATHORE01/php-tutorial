Introduction to Cookies

Cookies are small pieces of data stored on a user's computer by a web browser. They are commonly used in web development to store information that can be retrieved and used across different web pages.

Key Points:
  • Cookies are typically used to store user preferences, session data, and other information that needs to persist across multiple requests.
  • They are limited in size and can store a small amount of text-based data.
  • Cookies are often used for tasks like user authentication, remembering shopping cart items, and tracking user behavior.
  • Each cookie is associated with a specific domain and can be read by the web server that set it.
  • Cookies can have an expiration date, after which they are automatically deleted from the user's computer.
Setting Cookies in PHP:

In PHP, you can set cookies using the setcookie() function. Cookies are sent to the user's browser in the HTTP response headers.


// Set a cookie that expires in one hour
setcookie("user", "John", time() + 3600);
Retrieving Cookies in PHP:

To retrieve cookies in PHP, you can use the $_COOKIE superglobal array. It contains all the cookies sent by the user's browser.


// Retrieve the "user" cookie
$user = $_COOKIE["user"];
