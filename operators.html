PHP Operators

Operators in PHP are symbols that allow you to perform operations on variables and values. PHP provides various types of operators for different tasks. Here's a detailed overview of PHP operators:

Arithmetic Operators

Arithmetic operators are used for basic math operations. Common arithmetic operators include +, -, *, /, and %. For example:

<?php
    $x = 10;
    $y = 5;
    $sum = $x + $y;
    $product = $x * $y;
?>

Comparison Operators

Comparison operators are used to compare values. Common comparison operators include == (equal), != (not equal), < (less than), > (greater than), and more. For example:

<?php
    $a = 10;
    $b = 5;
    $result = ($a > $b);
?>

Logical Operators

Logical operators are used to perform logical operations. Common logical operators include and, or, not, and xor. For example:

<?php
    $isTrue = true;
    $isFalse = false;
    $result = $isTrue && $isFalse;
?>

Assignment Operators

Assignment operators are used to assign values to variables. Common assignment operators include =, +=, -=, and so on. For example:

<?php
    $x = 10;
    $x += 5; // $x is now 15
?>

String Operators

String operators are used to manipulate strings. The . operator is used for string concatenation. For example:

<?php
    $greeting = "Hello, ";
    $name = "John";
    $message = $greeting . $name;
?>

Ternary Operator

The ternary operator (conditional operator) allows you to assign a value based on a condition. For example:

<?php
    $age = 25;
    $message = ($age >= 18) ? "Adult" : "Minor";
?>

Understanding operators is essential in PHP, as they enable you to perform various operations on data and control the flow of your code. Different operators serve different purposes and are a fundamental part of PHP programming.
